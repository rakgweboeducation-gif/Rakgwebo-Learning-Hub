## Packages
framer-motion | For sophisticated page transitions and UI animations
recharts | For visualizing test results and progress
react-pdf | For rendering PDF textbooks
lucide-react | For beautiful iconography
date-fns | For date formatting in schedules and chats
react-day-picker | For calendar date selection in scheduling
clsx | For conditional class merging
tailwind-merge | For merging tailwind classes utility
react-hook-form | For robust form handling
zod | For schema validation in forms
@hookform/resolvers | For connecting Zod to React Hook Form
katex | For rendering mathematical equations (LaTeX)

## Notes
- PDF viewer requires setting up worker src correctly for react-pdf.
- Whiteboard will use HTML5 Canvas API with basic drawing features.
- Authentication uses a global context provider wrapping the app.
- Role-based routing is implemented in the main App component.
